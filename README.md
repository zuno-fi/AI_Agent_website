#ZunoFi - AI Agent_Website

Welcome to the AI Agent_Website repository by ZunoFi !

This project is focused on building a smart AI-powered financial recommendation platform designed specifically for college students and fresh graduates. The platform helps users make better money decisions by giving them personalized financial advice based on their income, expenses, and goals.

#Project Objective:

To develop a web-based AI Agent that:

Understands basic financial inputs from users
Provides actionable recommendations for budgeting, investing, and saving
Educates users on financial literacy in a conversational and beginner-friendly manner

#Target Audience:

College Students (18–24 years)
Fresh Graduates / Early Professionals (0–2 years of work experience)
Our goal is to simplify finance for Gen Z and first-time earners.

#Core Features:

Smart budgeting suggestions based on income and lifestyle
Investment recommendations (FDs, SIPs, equity, etc.)
Credit card and loan guidance tailored to user profile
Goal-based planning (e.g. saving for travel, emergency fund, gadgets)
Financial literacy tips through an interactive AI chatbot

#Tech Stack:

Frontend: React.js / Next.js (TBD)
Backend: Node.js / Express (TBD)
AI Layer: OpenAI GPT API
Database: MongoDB / Firebase (TBD)
Deployment: Vercel / Netlify (TBD)

#Folder Structure:

/frontend – UI and user interaction layer
/backend – API endpoints, business logic, AI integration
/docs – Architecture, design decisions, API documentation
/assets – Images, branding, and media files

#Getting Started:

Clone the repository
Install dependencies (npm install)
Run the development server (npm run dev)
Access the project on http://localhost:3000
